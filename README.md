Це командний проєкт, у якому реалізовано програму для створення, збереження та поступового доповнення текстового файлу. Кожен учасник команди виконує свій етап роботи з файлом, додаючи питання або відповіді. Програма демонструє навички роботи з файлами у Python — створення, запис, дописування та зчитування інформації.
